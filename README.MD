# CSY1063 – Portfolio Website

This repository contains my portfolio website developed for the module
**CSY1063 – Web Development**.

The website is created using **HTML5 and CSS3 only**, without the use of
external frameworks or libraries, in accordance with the assessment
requirements.

## Pages Included
- **index.html** – Home page introducing the portfolio and its purpose
- **project.html** – Projects page using a CSS Grid layout with placeholder projects
- **contact.html** – Contact page containing a mailto-based contact form
- **videoDemo.html** – Video demonstration page with a video link/embed
- **sitereport.html** – Site report including a reflective discussion, validation evidence, and references

## Features
- Responsive design with a mobile-only hamburger navigation menu
- Consistent layout and navigation across all pages
- CSS Grid used on the Projects page to demonstrate layout skills
- Subtle CSS hover animations for improved user experience
- Semantic HTML structure for accessibility and validation

## Repository Information
This repository demonstrates the use of **Git version control**, with multiple
commits made on different dates, as required by the assignment brief.

The website can be viewed locally by opening `index.html` in a web browser.
